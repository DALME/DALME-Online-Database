Model Templates (modelTemplates.py)¶

This file contains templates for models, used in dalme_app.models. These templates are used to more easily associate common data points with those models, things like timestamps and unique IDs.

class dalme_app.modelTemplates.dalmeBasic(*args, **kwargs)[source]¶

Model template with timestamps, but no pre-defined ID

class dalme_app.modelTemplates.dalmeIntid(*args, **kwargs)[source]¶

Model template with a unique ID assigned as a sequential integer

class dalme_app.modelTemplates.dalmeUuid(*args, **kwargs)[source]¶

Model template with a unique ID assigned by uuid.uuid4, resulting in a long, random identifier.

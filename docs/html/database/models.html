Models (models.py)¶

This file defines all of the models used in the application. These models are used to create database entries, and can be used in other functions to access and iterate through data in the database without writing SQL statements.

Deletes all external accounts associated with this account

pull_ids()[source]¶

For a given user, set external identifiers for associated accounts. These accounts have the same username as the associated user object
